## 概要
LINE広告（旧 Line Ads Platform）のレポートデータを転送する設定のDocsです。

## 設定項目
### STEP1 基本設定
| 項目名 | 必須 | 内容 |
| --- | --- | --- |
| LINE広告接続情報 | Yes | LINE広告の接続情報を選択してください。<br>詳しくは、[LINE広告の接続情報](/docs/connection-configuration-line-ad-api)を参照ください。 |
| 広告アカウントID | Yes |  [LINE Ad Manager](https://account.line.biz/login)の管理画面の**広告アカウント**タブから、データ取得対象の**広告アカウントID**を確認し、入力してください。<br>![image.png](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/data-source-line-ad-api-2024-08-29-21-43-0.png){height="" width=""}|
| ダウンロード種別 | Yes | **パフォーマンスレポート**、**マスターデータ(広告)** のいずれかを選択してください。 |

#### パフォーマンスレポート
| 項目名 | 必須 | 内容 |
| --- | --- | --- |
| レポート種別 | Yes | レポートの種類を選択します。<br>各レポートの概要については[データ種別](/docs/data-source-line-ad-api#データ種別)を、カラム単位の詳細については[カラム詳細](/docs/data-source-line-ad-api#カラム詳細)を参照ください。|
| データ取得期間 | Yes | レポートの抽出対象期間を`yyyy-MM-dd`形式で入力してください。 |
| カスタムコンバージョン | No | 取得したいカスタムコンバージョンがある場合は、こちらを入力してください。<br>カスタムコンバージョンについて、詳しくは[ウェブサイトのコンバージョンを計測する](https://www.linebiz.com/jp/manual/line-ads/tracking_016/)をご参照ください。 |
| 削除済みを含める | No | 「含める」を選択すると、削除されたキャンペーン、広告グループ、広告の情報を含んだレポートを取得できます。デフォルト値は「含めない」です。|

:::(Warning) (カスタムコンバージョン設定時のお願い)
既存の転送設定に対してカスタムコンバージョンの設定を追加・変更・削除した場合は、「自動データ設定の選択」にて**自動データ設定を実行**を選択してください。
![image](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/data-source-line-ad-api-2024-08-29-21-43-1.png){height="" width="500"}
:::

#### マスターデータ(広告)
| 項目名 | 必須 | 内容 |
| --- | --- | --- |
| マスターデータ種別 | Yes | マスターデータの種類を選択します。<br>各データの概要については[データ種別](/docs/data-source-line-ad-api#データ種別)を、カラム単位の詳細については[カラム詳細](/docs/data-source-line-ad-api#カラム詳細)を参照ください。|



## データ種別
### パフォーマンスレポート
| 種別 | レポート内容 |
| --- | --- |
| 広告アカウント | 広告アカウント単位の日別レポートを取得できます。 |
| キャンペーン | キャンペーン単位の日別レポートを取得できます。 |
| 広告グループ | 広告グループ単位の日別レポートを取得できます。 |
| 広告 | 広告単位の日別レポートを取得できます。 |

### マスターデータ(広告)
| 種別 | レポート内容 |
| --- | --- |
| キャンペーン | キャンペーンのマスターデータを取得できます。 |
| 広告グループ | 広告グループのマスターデータを取得できます。 |
| 広告 | 広告のマスターデータを取得できます。 |
