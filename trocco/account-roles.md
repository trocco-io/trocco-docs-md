## 概要

[ユーザーの権限設定](/docs/user-settings)では、ユーザーごとに以下のいずれかの**権限**を選択します。

- アカウント特権管理者
- アカウント管理者
- アカウントメンバー

ユーザーは、各権限に許可された操作を実行できます。
本ページでは、各権限によって「許可される操作」が異なる機能についてまとめています。

:::(Warning) (各種リソースの操作権限)

転送設定やデータマート定義などの各種リソースに対する操作権限は、[チーム機能](/docs/about-team)で管理されます。
アカウントに対する権限とは別の管理方法となりますので、ご注意ください。
ただし、[アカウント特権管理者](/docs/about-super-admin)に限り、自らにロールが付与されていないリソースであっても、操作できます。
:::

#### ユーザー管理

[ユーザー管理](/docs/about-account-user)に対する操作権限です。

![image](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/account-roles-2024-08-29-4-47-0.png){height="" width="500"}

| 操作＼権限 | アカウント特権管理者 | アカウント管理者 | アカウントメンバー |
| --- | --- | --- | --- |
| ユーザー管理画面の閲覧 | ✓ | ✓ | ✓ |
| 各ユーザーの権限設定の編集 | ✓ | ✓ | - |
| 他ユーザーが設定した**2要素認証**の無効化 | ✓ | - | - |
| ユーザーの招待 | ✓ | ✓ | - |
| ユーザーの削除 | ✓ | - | - |

#### アカウントセキュリティ

**アカウントセキュリティ**に対する操作権限です。

:::(Warning) (アカウントセキュリティ設定の影響範囲)

アカウントセキュリティ画面における各種設定はすべて、**アカウント全体**に適用されます。
設定を変更した場合、アカウント内のすべてのユーザーに影響を与えますので、変更する際はご注意ください。
:::

![image](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/account-roles-2024-08-29-4-47-1.png){height="" width="500"}

| 操作＼権限 | アカウント特権管理者 | アカウント管理者 | アカウントメンバー |
| --- | --- | --- | --- |
| **データプレビューの削除**の閲覧 | ✓ | ✓ | - |
| **データプレビューの削除**の有効化・無効化 | ✓ | ✓ | - |
| **SAML認証を使用したSSO**の閲覧 | ✓ | ✓ | - |
| **SAML認証を使用したSSO**の有効化・無効化 | ✓ | ✓ | - |
| **アクセス許可するIPアドレス**の閲覧 | ✓ | ✓  | ✓ |
| **アクセス許可するIPアドレス**の追加・編集・削除 | ✓ | ✓  | - |

#### チーム

[チーム](/docs/team)に対する操作権限です。

![image](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/account-roles-2024-08-29-4-47-2.png){height="" width="500"}

| 操作＼権限 | アカウント特権管理者 | アカウント管理者 | アカウントメンバー |
| --- | --- | --- | --- |
| チーム一覧画面の閲覧 | ✓ | ✓ | ✓ |
| チームの作成 | ✓ | ✓ | - |
| ロールを付与されていないチームの編集・削除 | ✓ | - | - |

:::(Warning) (チームの編集・削除権限)

チームを編集・削除するためには、該当チームに対するロールが必要になります。
詳しくは[チーム](/docs/team)を参照ください。
ただし、**アカウント特権管理者**に限り、自らにロールが付与されていないチームであっても、編集・削除できます。
:::
