転送設定一覧画面のDocsです。

## 絞り込み

条件を指定することで、転送設定を絞り込むことができます。
複数の条件を指定した場合はAND条件となり、より詳細な絞り込みができます。

| 項目名 | 内容 |
| --- | --- |
| 転送設定名 | 転送設定名を入力します。<br>入力した値に部分的に一致する転送設定名で絞り込みます。<br><br>**入力値の先頭と末尾を`/`にすると、正規表現を使用した転送設定名の絞り込みができます。**<br>詳しくは、[正規表現による転送設定名の絞り込み](/docs/etl-configuration-list#正規表現による転送設定名の絞り込み)を参照ください。 |
| コネクタ | 転送元と転送元のコネクタを選択します。<br>転送元・先のいずれかのコネクタのみでも絞り込みは可能です。 |
| 転送モード | 転送先がDWH系コネクタの場合に利用できます。<br>転送モードを選択します。 |
| ラベル | ラベルを選択します。<br>複数選択が可能です。 |
| 作成者 | 転送設定の作成者を選択します。<br>「作成者が自分」にチェックを入れると、自分が作成した転送設定で絞り込めます。 |
| 接続情報 | 接続情報名を選択します。<br>[閲覧者以上のロール](/docs/resource-group)を持つ接続情報でのみ、絞り込みできます。 |
| 環境 | 環境グループ・環境を選択します。 |
| スケジュール設定 | スケジュール設定の有無を選択します。<br>「設定あり」の場合は、スケジュールのON・OFF状況で絞り込めます。 |
| 通知設定 | 通知設定の有無を選択します。 |

### 正規表現による転送設定名の絞り込み

**転送設定名**の入力値の先頭と末尾を`/`にすると、転送設定名の絞り込み条件を正規表現で指定できます。
使用できる構文は、[MySQL 8.0 リファレンスマニュアル | 正規表現構文](https://dev.mysql.com/doc/refman/8.0/ja/regexp.html#regexp-syntax)を参照ください。
なお、構文で使用される`^`や`*`などのメタ文字を絞り込み対象の文字列にする場合、`\^`や`\*`のようにバックスラッシュ（`\`）でエスケープする必要があります。

#### 入力例

- `/^perfect_match_word$/`
  - この場合、`perfect_match_word`と完全に一致する転送設定名が対象になります。
- `/^begin_word.*end_word$/`
  - この場合、`begin_word`で始まり`end_word`で終わる転送設定名が対象になります。
- `/^meta_word\(abc\^\)$/`
  - この場合、`meta_word(abc^)`と完全に一致する転送設定名が対象になります。
