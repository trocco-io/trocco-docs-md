本ページでは、バクラクに接続するための設定（接続情報）について説明します。
転送に関する設定については、[転送元 - バクラク](/docs/data-source-bakuraku)を参照ください。

## 認証方式

- API Key

## 利用上の注意・制約

:::(Warning) (API利用制限)

バクラクAPIには利用制限があります。短時間に大量のAPIリクエストを送信すると、一時的にAPIが使用できなくなる可能性があります。
:::

## 設定項目

|項目名|必須|説明|
|---|---|---|
|API Key|✓|バクラクで発行したAPI Keyを入力します。|

## バクラク側で必要な作業

### API Keyの取得

1. バクラクの管理画面にログインします
2. 「API管理」メニューにアクセスします
3. 「新規作成」をクリックして、必要な権限を選択し、新しいAPI Keyを発行します
4. 発行されたAPI Keyをコピーして、TROCCO側の設定に使用します

### 権限の付与

#### 転送元に利用する場合

データを取得するために、API Keyに対して以下の権限が必要です。

- 申請の読み取り権限
- 組織図の読み取り権限
- ユーザーの読み取り権限

## 困ったときには・トラブルシューティング

### 認証エラーが発生する

#### 考えられる原因

- API Keyが無効または期限切れ
- API Keyの権限が不足している
- APIレート制限に達している

#### 対処方法

1. API Keyが正しく入力されているか確認してください
2. バクラクの管理画面でAPI Keyが有効であることを確認してください
3. 必要な権限がAPI Keyに付与されているか確認してください
4. APIレート制限に注意し、適切な間隔で実行してください
