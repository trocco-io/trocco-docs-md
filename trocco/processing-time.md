## 概要

[チーム機能](/docs/about-team)をご利用の場合、リソースグループおよび各ジョブの種類ごとに、各ジョブの処理時間や合計処理時間を確認できます。
これらのデータを活用することで、たとえば処理時間が多いリソースグループを特定して調整するなど、TROCCO の運用面における効率化・最適化を図ることができます。
なお、リソースグループに紐づけられていないジョブについては、「リソースグループに属さないリソース」として集計し表示されます。

:::(info) (処理時間機能のご利用について)
処理時間機能を利用できるのはチーム機能オプションを有効にしているお客様に限ります。
:::

:::(Info) (処理時間の計算に使われるタイムゾーンについて)
タイムゾーンについては、お客様の設定のタイムゾーン設定にかかわらず日本時間にて集計されます
:::

:::(Info) (削除済みのリソースグループについて)
ジョブ実行時点でジョブ定義に紐づけられていたリソースグループがその後削除された場合、本画面においては「削除済みリソースグループ」として集計し表示されます
:::

:::(Warning) (処理時間機能のデータについて)
処理時間機能に反映されるデータは 2024 年 11 月初日以降のデータのみです。
:::

## 各ラベルの説明

| 列名                       | 説明                                                                                                                             |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| リソースグループ名         | 集計対象となるリソースグループ名です                                                                                             |
| 合計処理時間               | 集計対象となるリソースグループに紐づくジョブ実行により消費された処理時間です                                                     |
| 合計処理時間比率           | 合計処理時間を月のご契約の処理時間で割った比率です                                                                               |
| データ転送処理時間         | 該当リソースグループに紐づくデータ転送ジョブの処理時間のうち、リソース増強オプションを設定していない転送ジョブの処理時間合計です |
| データ転送処理時間(Large)  | 該当リソースグループに紐づくデータ転送ジョブの処理時間のうち、リソースサイズを `Large` に設定した転送ジョブの処理時間です (\*1)  |
| データ転送処理時間(XLarge) | 該当リソースグループに紐づくデータ転送ジョブの処理時間のうち、リソースサイズを `XLarge` に設定した転送ジョブの処理時間です (\*1) |
| データマート処理時間       | 該当リソースグループに紐づくデータマートジョブの処理時間合計です                                                           |
| dbt 処理時間               | 該当リソースグループに紐づく dbt 連携ジョブの処理時間合計です                                                                    |
| ワークフロータスク処理時間 | 該当リソースグループに紐づくワークフロータスクの処理時間合計です (\*2)                                                           |

\*1 プロフェッショナルプランをご契約いただいていないお客様の場合、固定で `00m00s` の値が表示されます。
\*2 転送ジョブ、データマートジョブ、dbt 連携ジョブは、ワークフロー経由から実行された場合であってもそれぞれのジョブの集計対象として計算されます。
