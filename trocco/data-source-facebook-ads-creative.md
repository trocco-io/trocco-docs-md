## 概要
Facebook広告のクリエイティブデータを取得・転送する設定のDocsです。

## 制約
* アクセストークン失効などの原因により転送が失敗する場合がございます。[接続情報 - Facebook](/docs/connection-configuration-facebook-ad-insights)を参考に再認証をお願いします。
* アクセストークンは60日ごとに再認証が必要となります。

## 設定項目
### STEP1 基本設定
| 項目名 | 必須 | デフォルト値 | 内容 |
| --- | --- | --- | --- |
| Facebook接続情報 | Yes | - | あらかじめ登録してある接続情報から、今回の転送設定に必要な権限を持つものを選択します。 |
| 広告アカウントID | Yes | - | クリエイティブデータを取得する広告アカウントIDを指定します。（※1） |
| 作成日時 | NO | - | データを取得する範囲の作成日時（カラム名:created_time）を指定します。<br>取得されるデータは開始/終了日時を含む範囲のデータになります。<br><br>例えば2021/06/30の一日分のデータを取得する場合、<br>開始日時: 2021-06-30 00:00:00<br>終了日時: 2021-06-30 23:59:59<br>を指定してください。<br><br>タイムゾーンが未指定の場合、Asia/Tokyoの日時として扱われます。 |
| 更新日時 | NO | - | データを取得する範囲の更新日時（カラム名:updated_time）を指定します。<br>取得されるデータは開始/終了日時を含む範囲のデータになります。<br><br>例えば2021/06/30の一日分のデータを取得する場合、<br>開始日時: 2021-06-30 00:00:00<br>終了日時: 2021-06-30 23:59:59<br>を指定してください。<br><br>タイムゾーンが未指定の場合、Asia/Tokyoの日時として扱われます。 |

※1 広告アカウントIDの確認方法は、Facebook for Businessの「[Facebook広告アカウントのID番号を見つける](https://www.facebook.com/business/help/1492627900875762)」をご参照ください。
