## 概要
Slackの接続情報を設定する際のDocsとなります。

## 入力項目
| 項目名 | 必須 | 内容 |
| --- | --- | --- |
| 名前 | Yes | TROCCO内部で利用する接続情報の名前を入力してください。 |
| メモ | No | TROCCO内部で利用する接続情報のメモを入力してください。 |
| API Token | Yes | SlackのAPIトークンを入力してください。<br>取得方法は[公式ドキュメント](https://api.slack.com/legacy/custom-integrations/legacy-tokens)をご参照下さい。 |
