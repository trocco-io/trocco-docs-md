本ページでは、マネーフォワード クラウド会計からデータを取得するための設定（転送設定）について説明します。
接続に関する設定については、[接続情報 - マネーフォワード クラウド会計](/docs/connection-configuration-money-forward-cloud-accounting)を参照ください。


## 利用上の注意・制約

:::(Warning) (API レート制限)

マネーフォワード クラウド会計 API では、各エンドポイントに対して一定頻度のアクセス制限が設定されています。
制限に達すると、HTTPステータスコード **429 (Too Many Requests)** が返され、転送が一時的に停止します。

大量のデータを転送する場合は、実行時間に余裕を持って設定してください。
:::

:::(Info) (API 使用可能プランについて)

マネーフォワード クラウド会計 API を利用するには、API使用可能なプラン（有料プランなど）の契約が必要です。
詳しくは、マネーフォワード クラウドの公式サイトをご確認ください。
:::

## 事前準備

### 接続情報の設定

マネーフォワード クラウド会計からデータを取得するには、事前に接続情報の設定が必要です。
詳しくは、[接続情報 - マネーフォワード クラウド会計](/docs/connection-configuration-money-forward-cloud-accounting)を参照ください。

## 取得対象


### 勘定科目一覧

勘定科目一覧のデータを取得します。

#### 設定項目

|項目名|必須|デフォルト値|説明|
|---|---|---|---|
|**有効**|✓|全て|「全て」か「有効のみ」かを指定します。|

#### 取得データ

|カラム名|デフォルトのデータ型|補足事項|
|---|---|---|
|`id`|`string`|勘定科目ID|
|`financial_statement_type`|`string`|決算書科目|
|`name`|`string`|勘定科目名称|
|`available`|`boolean`|有効|
|`tax_id`|`string`|税区分ID|
|`search_key`|`string`|検索キー|
|`sub_accounts`|`json`|サブ勘定科目（SubAccount）の配列|
|`account_group`|`string`|勘定科目グループ|

|`voucher_file_ids`|`json`|添付された証憑IDの配列|
