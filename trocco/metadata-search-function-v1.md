## 概要

TROCCOのデータカタログでは、テーブルおよびカラムを**メタデータも踏まえて**検索できます。
また、検索対象や検索範囲の絞り込みにも対応しています。

:::(Info) (あいまい検索にも対応)

検索エンジンとして**Elasticsearch**を採用しており、あいまい検索にも対応しています。
たとえば「売上」または「売り上げ」と検索したときに、いずれも同じ検索結果が得られます。
:::

## 検索対象文字列

以下のいずれかに含まれる文字列が検索対象となります。

- テーブル・カラム名、またはテーブルが所属している各要素の名前
  - プロジェクト / データベース名
  - データセット / スキーマ名
  - テーブル名
  - カラム名
- 基本メタデータ
  - 論理名
  - 説明
- ユーザー定義メタデータ
  - `String`型フィールドの入力内容
  - `Text(Markdown)`型フィールドの入力内容
- 関連する転送設定
  - 関連する転送設定名
  - 関連する転送設定メモ
- <details>
   <summary>データソースに関するメタデータ</summary>

  - kintoneアプリ名
  - kintoneフィールド名
  - kintone説明
  - MySQLカラム名
  - MySQLテーブル名
  - Salesforceオブジェクト名
  - Salesforceフィールド名
  - Salesforceヘルプテキスト
  - Salesforce説明
  - Salesforce日本語列名
  - Yahoo!検索広告フィールド日本語表示名
  - Yahoo!検索広告レポートフィールド名
</details>

## 検索方法

検索バーに検索したい文字列を入力し、虫眼鏡アイコンをクリック（またはEnterキーを押す）と、以下のように検索結果が表示されます。

### 検索結果の絞り込み

上記の検索は、すべてのテーブル・カラムが検索対象となります。
検索結果は、以下の通り絞り込むことができます。

- 表示対象の絞り込み
  - デフォルトでは、すべてのテーブル・カラムが検索対象となります。
  - 検索対象を絞り込みたい場合は、適宜チェックボックスを適宜外してください。
- 検索範囲の絞り込み
  - デフォルトでは、すべての**プロジェクト**（またはデータベース）が検索対象となります。
  - 検索範囲を絞り込みたい場合は、**プロジェクト**（またはデータベース）を選択してください。
  - **プロジェクト**（またはデータベース）を選択すると、その下位階層の選択項目が順次表示され、より細かく絞り込むことができます。
