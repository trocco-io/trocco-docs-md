## 概要

SHANON MARKETING PLATFORMの接続情報を設定する際のDocsとなります。

## 入力項目

| 項目名 | 必須 | 内容 |
| --- | --- | --- |
| 名前 | ✓ | TROCCO内部で利用する接続情報の名前を入力してください。 |
| メモ | - | TROCCO内部で利用する接続情報のメモを入力してください。 |
| マーケティングプラットフォームAPIキー | ✓ | 接続に使用するAPIキーを入力します。<br>[入力項目の取得方法と入力手順](/docs/connection-configuration-shanon-marketing-platform#入力項目の取得方法と入力手順)を参照ください。 |
| パスワード | ✓ | 接続に使用するパスワードを入力します。<br>[入力項目の取得方法と入力手順](/docs/connection-configuration-shanon-marketing-platform#入力項目の取得方法と入力手順)を参照ください。 |
| 秘密鍵 | ✓ | 接続に使用する秘密鍵を入力します。<br>[入力項目の取得方法と入力手順](/docs/connection-configuration-shanon-marketing-platform#入力項目の取得方法と入力手順)を参照ください。 |
| ドメイン名 | ✓ | SHANON MARKETING PLATFORM管理画面のURLのドメインを入力してください。（例：`https://xxxxxx.smktg.jp/`の場合は「`xxxxxx.smktg.jp`」の部分） |

### 入力項目の取得方法と入力手順

**マーケティングプラットフォームAPIキー**と**パスワード**と**秘密鍵**を、下記の手順で取得し入力してください。

1. SHANON MARKETING PLATFORM管理画面にログインします。
2. 管理画面の**設定**>**API**をクリックします。
   ![image.png](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/connection-configuration-shanon-marketing-platform-2024-08-29-9-55-0.png){height="" width="400"}
3. **新規登録**をクリックします。
   ![image.png](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/connection-configuration-shanon-marketing-platform-2024-08-29-9-55-1.png){height="" width="600"}
4. 各種項目を入力して、**登録**をクリックします。
   ![image.png](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/connection-configuration-shanon-marketing-platform-2024-08-29-9-55-2.png){height="" width="600"}
5. 表示される情報を、対応する項目に入力します。
  ![image.png](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/connection-configuration-shanon-marketing-platform-2024-08-29-9-55-3.png){height="" width="600"}
    - APIキー：接続情報の**マーケティングプラットフォームAPIキー**
    - パスワード：接続情報の**パスワード**
    - シークレットキー：接続情報の**秘密鍵**
