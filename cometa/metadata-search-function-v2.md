## 概要

COMETAでは、各種アセットおよび用語を**メタデータも踏まえて**検索できます。
また、検索対象や検索範囲の絞り込みにも対応しています。

![metadata-search-function-v2-2024-08-29-1-6-0](https://cdn.document360.io/3a3bae1e-f157-487f-8798-01e9d820e760/Images/Documentation/metadata-search-function-v2-2024-08-29-1-6-0.png){height="" width="50%"}

:::(Info) (あいまい検索・セマンティック検索)

検索エンジンとして**Elasticsearch**を採用しており、あいまい検索にも対応しています。
たとえば「売上」または「売り上げ」と検索したときに、いずれも同じ検索結果が得られます。

また、検索対象が用語の場合は、意味が近しいワードもヒットするセマンティック検索にも対応しています。
:::

## 検索対象文字列

以下のいずれかに含まれる文字列が検索対象となります。
この検索対象文字列は、[AIチャット機能](/cometa/docs/ai-assist)でも共通して使用されます。

### データベース系データストアの場合

[データベース系データストア](/cometa/docs/about-data-catalog-v2#データベース系データストア)の検索対象文字列は以下のとおりです。

- テーブル・カラム名、またはテーブルが所属している各要素の名前
  - プロジェクト / データベース名
  - データセット / スキーマ名
  - テーブル名
  - カラム名
- 基本メタデータ
  - 論理名
  - 説明
- ユーザー定義メタデータ
  - `String`型フィールドの入力内容
  - `Text(Markdown)`型フィールドの入力内容
- dbtの説明

:::(Info) (TROCCOと合わせてご利用の場合)

TROCCOと合わせてご利用の場合は、上記に加えて、自動取得される**転送設定に関する情報**および[**データソースに関するメタデータ**](/cometa/docs/automatic-metadata-fetched-v2#データソースに関するメタデータ)に含まれる文字列も検索対象となります。

<details>
   <summary>追加で検索対象文字列となる一覧</summary>

- 関連する転送設定名
- 関連する転送設定メモ
- kintoneアプリ名
- kintoneフィールド名
- kintone説明
- MySQLカラム名
- MySQLテーブル名
- Salesforceオブジェクト名
- Salesforceフィールド名
- Salesforceヘルプテキスト
- Salesforce説明
- Salesforce日本語列名
- Yahoo!検索広告フィールド日本語表示名
- Yahoo!検索広告レポートフィールド名
</details>
:::

### ダッシュボード系データストアの場合

[ダッシュボード系データストア](/cometa/docs/about-data-catalog-v2#ダッシュボード系データストア)の検索対象文字列は以下のとおりです。

- アセット名（TableauワークブックやTableauダッシュボードなどの名前）
- プロジェクト名
- 説明

### 用語の場合

[用語](/cometa/docs/glossary-and-term_v2)の検索対象文字列は以下のとおりです。

- 用語名
- 説明文

## 検索方法

検索バーに検索したい文字列を入力し、虫眼鏡アイコンをクリックする（またはEnterキーを押す）と、検索結果が表示されます。

### 検索結果の絞り込み

データベース・ダッシュボードの場合、上記の検索は連携済みのすべてのデータストアの配下にあるすべてのアセットが検索対象となります。
検索結果は、以下の通り絞り込むことができます。

- 表示対象の絞り込み
  - デフォルトでは、すべてのアセットが検索対象となります。
  - 検索対象のアセットを絞り込みたい場合は、適宜チェックボックスを適宜外してください。
- 検索範囲の絞り込み
  - デフォルトでは、連携済みのすべてのデータストアが検索対象となります。
  - 検索範囲を絞り込みたい場合は、データストアを選択してください。
  - データストアを選択すると、その下位階層の選択項目が順次表示され、より細かく絞り込むことができます。
